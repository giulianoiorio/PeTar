{\footnotesize\ttfamily bool Timestep\+::check\+\_\+repeat (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{property_8h_a44145f6f9805df57da735ea6ac0f1dbb}{\+\_\+\+U\+N\+U\+S\+ED}} \mbox{\hyperlink{class_star}{Star}} $\ast$}]{s,  }\item[{const double \&}]{m0,  }\item[{const double \&}]{m,  }\item[{const double \&}]{derivative,  }\item[{const double}]{tstart }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}

This function check if the current evolution time step has caused a too fast evolution of a property with values {\ttfamily m0} (before the evolution), {\ttfamily m} (after the evolution) and time derivative ({\ttfamily derivative}). If the property is related to some subphase (e.\+g. \mbox{\hyperlink{class_m_h_e}{M\+HE}}, \mbox{\hyperlink{class_m_c_o}{M\+CO}}) we should also set the initial time of this phase with the parameter {\ttfamily tstart}. If the property has not subphases (e.\+g. the \mbox{\hyperlink{class_radius}{Radius}}) {\ttfamily tstart} should be set to 0. At the beginning of the function the new\+\_\+dt is set to the max dt allowed (Delta\+\_\+t phase/starparameter\+::min\+\_\+points\+\_\+per\+\_\+phase). Then if tstart is 0 the function check if the relative variation ({\ttfamily m} -\/{\ttfamily m0})/ {\ttfamily m} is lower that 2 times \mbox{\hyperlink{namespacestarparameter_a8398de803188e71e7b07a21cd0c78d0e}{starparameter\+::maximum\+\_\+variation}} (default 0.\+05). If it is lower return false, else set a new dt to half of the current dt and set repeat to true. if tstart is not 0 there are different options\+: if m0=0 and m$>$\mbox{\hyperlink{namespacestarparameter_a8398de803188e71e7b07a21cd0c78d0e}{starparameter\+::maximum\+\_\+variation}}\+: set a new\+\_\+dt as 0.\+5$\ast$(\mbox{\hyperlink{class_localtime}{Localtime}} -\/ tstart) + (tstart -\/ Localtime\+\_\+0) (time from Localtime\+\_\+0 to start + half of Localtime-\/tstart) if the realtive variaiton ((\mbox{\hyperlink{class_localtime}{Localtime}} + new\+\_\+dt)-\/tstart)/tstart $<$1e-\/2 do not repeat else repeat with the new dt (return immediatly). if m0$>$2 and relative difference (m-\/m0)/m0 $<$ 2$\ast$starparameter\+::maximum\+\_\+variation not needed to repeat else set repeat to true and set new\+\_\+dt=\mbox{\hyperlink{namespacestarparameter_a8398de803188e71e7b07a21cd0c78d0e}{starparameter\+::maximum\+\_\+variation}}$\ast$({\ttfamily m/{\ttfamily derivative})} if m02$<$2 and absolute difference (m-\/m0) $<$4$\ast$starparameter\+::maximum\+\_\+variation not needed to repeat else set repeat to true and new\+\_\+dt=2.\+0$\ast$starparameter\+::maximum\+\_\+variation$\ast$({\ttfamily m/{\ttfamily derivative})} 

Finally there is a final check, if the new\+\_\+dt is larger than 0.\+8$\ast$dt return False.

Evolve the \mbox{\hyperlink{class_timestep}{Timestep}}, the new value V will be the dt to use in the next iteration. The function passes through the following steps\+:
\begin{DoxyItemize}
\item Check if the \mbox{\hyperlink{class_m_h_e}{M\+HE}}, \mbox{\hyperlink{class_m_c_o}{M\+CO}} or the \mbox{\hyperlink{class_radius}{Radius}} have evolved too much (comparing with \mbox{\hyperlink{namespacestarparameter_a8398de803188e71e7b07a21cd0c78d0e}{starparameter\+::maximum\+\_\+variation}}). If this happened the function set repeat to True and set a new dt (see \mbox{\hyperlink{class_timestep_af3a9abc90c105526d4e650638a025291}{Timestep\+::check\+\_\+repeat}})
\item If not needed to repeat determine next dt by imposing a maximum relative variation of stellar mass and radius.
\item Check if the dt should be changed because we are close to phase change (e.\+g. formation of the He core, CO core, remnant, ...)
\begin{DoxyItemize}
\item Set reference\+\_\+time to 0 if \mbox{\hyperlink{class_localtime}{Localtime}} + new\+\_\+dt$<$time\+\_\+next\+\_\+phase else reference\+\_\+time=time\+\_\+next\+\_\+phase
\item if reference\+\_\+time is 0, new\+\_\+dt=min(new\+\_\+dt, 0.\+999$\ast$(reference\+\_\+time -\/ \mbox{\hyperlink{class_localtime}{Localtime}}) ) else new\+\_\+dt=min(new\+\_\+dt, 1.\+001 $\ast$(reference\+\_\+time -\/\mbox{\hyperlink{class_localtime}{Localtime}}).
\end{DoxyItemize}
\item Finally set \mbox{\hyperlink{class_property_a3b81e1e12799a9796befb8f4edcb0dd7}{Timestep\+::V}} to new\+\_\+dt
