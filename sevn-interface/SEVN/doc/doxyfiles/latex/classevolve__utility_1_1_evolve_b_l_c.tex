\doxysubsection{Detailed Description}
Evolve functor specialised for the analysis of stages of binary lifetime with a compact object plus a luminous variable (B\+LC). The evolution starts record the binary states (following the dtout option) when we enter in a phase with an unbroken binary containing a compact object plus a luminous star (\mbox{\hyperlink{class_phase}{Phase}}$<$7 or WD). The evolution is broken when a broken trigger is true or after we conclude a B\+LC phase since there is no way that a new B\+LC phase can occur. @\+Note The dtout end option means that only the start and the end of the B\+LC phase are printed; @\+Note This evolution can be used only with binaries, using it with stars raise a not implemented error. Evolve functor specialised for the analysis of stages of binary lifetime with a compact object plus a luminous variable (B\+LC). @record\+\_\+state\+\_\+policy The states start to be recorded when the binary enters the first time in a B\+LC phase (bound compact + Luminous star) The last record is got the first time the binary exits the B\+LC phase (e.\+g. the Luminous star becomes a compact object or a merger occurs). Inside this window the state are recorded following the dtout option.
