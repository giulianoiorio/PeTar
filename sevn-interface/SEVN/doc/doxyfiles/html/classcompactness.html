◆ triggering_explosion()

bool compactness::triggering_explosion ( double  csi25)
protected

Estimate if the SN trigger an explosion or an implosion

Parameters
csi25Compacteness parameters csi25
Returns
true if an explosion is trigerred, false if an implosion is triggered @Note the estimate method depends on the parameter csi25_explosion_tshold if it is positive we just compare csi25 with csi25_explosion_tshold. If csi25_explosion_tshold=-1, we estimate the likelihood of explosion at given csi25 (from Patton&Sukhbold20) and then we draw a random number between 0 aand 1, if it is lower than per probability lkelihood a explosion is triggered.
