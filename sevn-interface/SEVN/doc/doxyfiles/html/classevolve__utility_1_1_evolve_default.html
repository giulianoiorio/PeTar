Detailed Description

Default evolve. Same evolve for both stars and binstars (see method evolve_default). @record_state_policy Following the dtout option in input dtout=end record only the first and last state of the binary @break_evolve_policy The evolution is stopped when a break trigger is set to true (depening on the tf option in input). @binstar_evolve Yes @star_evolve Yes @catch sevnerr
