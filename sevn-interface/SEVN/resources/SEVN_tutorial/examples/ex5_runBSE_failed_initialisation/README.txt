Compile SEVN if needed (./compile.sh, Insert the correct SEVN path inside compile.sh)
Insert the correct SEVN path inside run.sh
Run SEVN with ./run.sh 

You will notice that some of the systems will not be evolved and they end in the file failed_initialisation.txt. These are stars that have been not initialised due some errors/inconsistencies in the input parameter. 
Try to understand what is the problem with the not-initialised stars looking at the file failed_initialisation.txt
